üíæ 1. Rangkuman: Sumber Data

Analisis data dimulai dari pengumpulan data. Sumber data secara umum terbagi dua:

üìù Data Primer

Data yang dikumpulkan langsung oleh peneliti dari sumber pertama.

Metode:

  • Wawancara
  • Survei / Kuesioner
  • Observasi (Pengamatan)

üìö Data Sekunder

Data yang sudah ada dan dikumpulkan oleh pihak lain.

Sumber:

  • Data BPS (Badan Pusat Statistik)
  • Jurnal ilmiah atau laporan
  • Database perusahaan

‚öôÔ∏è 2. Rangkuman: Pengolahan Data

Setelah dikumpulkan, data mentah harus diolah. Ada dua cara utama:

Pengolahan Data Manual

Dilakukan tanpa komputer, biasanya menggunakan alat sederhana seperti kalkulator dan turus (tally) untuk menghitung frekuensi.

Contoh Manual:

Mendata 5 nilai: 7, 8, 7, 9, 8. Menghitung Rata-rata: (7+8+7+9+8) / 5 = 39 / 5 = 7.8. Metode ini lambat dan rentan salah hitung jika datanya banyak.

Pengolahan Data dengan Komputer

Menggunakan software spreadsheet seperti Microsoft Excel atau Google Sheets. Metode ini cepat, akurat, dan bisa mengolah jutaan data.

Fungsi utamanya:

  • Membuat Tabel: Mengatur data dalam baris dan kolom.
  • Menghitung Statistik: Menggunakan rumus seperti =SUM() (total), =AVERAGE() (rata-rata), =MAX() (tertinggi), =MIN() (terendah).
  • Membuat Grafik: Visualisasi data instan (Grafik Batang, Garis, Lingkaran).

üéÆ Latihan: Simulasi Pengolahan Komputer

Hitung & Tafsirkan Data

Masukkan nilai-nilai (pisahkan dengan koma) untuk melihat hasil statistik dan interpretasinya secara otomatis.

üí° 3. Rangkuman: Penafsiran & Pengambilan Keputusan

Ini adalah langkah terpenting: mengubah data (angka) menjadi informasi (makna) dan tindakan (keputusan).

Menafsirkan Data

Bukan hanya tahu angkanya, tapi tahu artinya.

  • Menafsirkan Statistik: "Rata-rata (Mean) nilai kelas adalah 85" berarti performa kelas secara umum baik. "Nilai terendah (Min) adalah 40" berarti ada siswa yang butuh perhatian.
  • Menafsirkan Grafik: "Grafik garis penjualan naik" berarti tren positif. "Grafik batang produk A lebih tinggi dari B" berarti produk A lebih laku.

Pengambilan Keputusan

Menggunakan penafsiran untuk mengambil langkah selanjutnya.

Contoh Alur:

Fakta (Data): Grafik penjualan "Siomay" paling rendah di kantin.
Penafsiran (Informasi): Siomay tidak diminati siswa.
Keputusan (Tindakan): Mengganti menu Siomay dengan menu lain, atau membuat promosi diskon Siomay.
