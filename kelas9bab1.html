Berpikir Komputasional adalah cara berpikir untuk memecahkan masalah dengan menggunakan konsep dasar ilmu komputer. Ini bukan hanya tentang coding, tapi tentang bagaimana kita mengurai masalah menjadi bagian-bagian kecil yang bisa diselesaikan secara sistematis!

üéØ 4 Pilar Berpikir Komputasional:

1. Dekomposisi: Memecah masalah besar menjadi bagian kecil
2. Pattern Recognition: Mencari pola atau kesamaan
3. Abstraksi: Fokus pada informasi penting, abaikan detail tidak penting
4. Algoritma: Membuat langkah-langkah solusi yang sistematis

üìù Algoritma dan Pemrograman

Algoritma adalah urutan langkah-langkah logis untuk menyelesaikan suatu masalah. Seperti resep masakan, algoritma memberikan instruksi yang jelas dan berurutan!

üìã Algoritma

Langkah-langkah sistematis untuk menyelesaikan masalah. Contoh: cara membuat mie instan!

üíª Pemrograman

Menerjemahkan algoritma ke dalam bahasa yang dimengerti komputer.

üîÑ Debugging

Mencari dan memperbaiki kesalahan dalam program kita.

üìö Struktur Kode Robomind

Robomind menggunakan bahasa pemrograman yang sederhana dengan perintah-perintah dasar yang mudah dipahami:

üéÆ Perintah Dasar Robomind:

// Perintah Gerakan
forward(n) // Maju n langkah
backward(n) // Mundur n langkah
left() // Belok kiri 90 derajat
right() // Belok kanan 90 derajat

// Perintah Aksi
pickUp() // Ambil objek
putDown() // Letakkan objek
paintWhite() // Cat putih

// Perintah Kontrol
repeat(n) { } // Ulangi n kali
if(...) { } // Percabangan kondisi
while(...) { } // Perulangan dengan kondisi

üß© Uji Pemahaman Materi

Jawablah 5 pertanyaan berikut untuk menguji pemahamanmu tentang Berpikir Komputasional dan Robomind!

Pertanyaan 1: Analisis Konsep

Seorang siswa ingin membuat robot yang bisa mengambil semua bola di ruangan dan meletakkannya di satu tempat. Jelaskan bagaimana kamu akan menggunakan 4 pilar berpikir komputasional (Dekomposisi, Pattern Recognition, Abstraksi, dan Algoritma) untuk menyelesaikan masalah ini!

Pertanyaan 2: Pemecahan Masalah

Perhatikan kode Robomind berikut:

forward(3)
right()
forward(2)
left()
forward(3)

Jika robot menghadap ke utara di awal program, ke arah mana robot akan menghadap di akhir program? Gambarkan jalur pergerakan robot dan jelaskan alasanmu!

Pertanyaan 3: Berpikir Logis

Mengapa perulangan (loop) sangat penting dalam pemrograman? Berikan contoh kasus nyata dalam kehidupan sehari-hari yang bisa dianalogikan dengan konsep perulangan, dan jelaskan bagaimana hal itu bisa diterapkan dalam program Robomind!

Pertanyaan 4: Debugging & Problem Solving

Seorang programmer membuat kode untuk robot membentuk persegi, tetapi hasilnya malah membentuk segitiga. Apa saja kemungkinan kesalahan yang bisa terjadi? Bagaimana cara kamu melakukan debugging untuk menemukan dan memperbaiki error tersebut?

Pertanyaan 5: Aplikasi & Kreativitas

Bayangkan kamu harus membuat program Robomind untuk robot yang bisa menyelesaikan maze (labirin). Buatlah pseudocode atau algoritma sederhana yang menjelaskan logika robot untuk menemukan jalan keluar. Pertimbangkan: Bagaimana robot mendeteksi dinding? Apa strategi terbaik agar robot tidak berputar-putar di tempat yang sama?
